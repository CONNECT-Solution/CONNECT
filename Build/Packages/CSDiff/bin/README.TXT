*** Introduction ***
ComponentSoftware Diff (CSDiff) is a free, advanced file-difference analysis tool for Microsoft Windows. CSDiff is a great asset to anyone in need of analyzing changes which have been made between two revisions of the same file or folder. CSDiff is commonly used when analyzing program source files, HTML documents and MS-Word documents.

Key features of CSDiff:
 Powerful folder and single file comparison tool.
 Single screen display provides a unique, intuitive difference analysis, making it ideal for printout and reports.
 Edit mode allows to accept, reject or manually edit any modifications.
 Change detection by line, by word or by character.
 Smart white-space difference analysis option. 
 Analysis of MS-Word documents. (MS-Word required)
 Sophisticated folder difference analysis sorted by any key.
 Support of compressed folders (ZIP files) for comparison between archived folders and regular folders.
 Enhanced folder difference analysis reports, including: HTML reports, XML reports, filters and optional links to file-level analysis. 
 Freeware -- CSDiff may be used completely free of charge.

For CSDiff tips and tricks and for new version information, visit the CSDiff home page at: http://www.ComponentSoftware.com/products/csdiff/

CSDiff is part of CS-RCS -- GNU RCS compatible revision control system that is fully integrated with Windows 95/NT. CS-RCS is free for individuals. For full details, visit CS-RCS home page at: http://www.ComponentSoftware.com/products/rcs/

*** Installation and Activation ***

Uninstall any previously installed version of CSDiff.
Run CSDiff42.exe from a temporary folder and follow the instructions in order to install CSDiff to the desired location. 
(e.g., ${environment::get-variable('ProgramFiles')}\CSDiff)


*** To uninstall CSDiff ***

Run Uninstall.exe from your CSDiff program folder in order for the program and all its components to be uninstalled
or
click Start, Settings and Add/Remove programs and select CSDiff from the list of installed programs.


*** Requirements ***

Windows 95/98/Me/XP or Windows NT/2000 (NT 3.51 & 4.0 or later).
Analysis of MS-Word documents requires MS-Word.
