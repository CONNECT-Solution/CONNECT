Future Enhancements

Here is a sample of some features planned for the near future:

  • Add option -ignore:FILE to ignore fragments of code
  • Add processing time to HTML report (Benoit Xhenseval).
  • Fix incorrect % duplication in HTML report (Benoit Xhenseval).
  • Nant task
  • Ruby support: handle regular expressions, ?x character literals and
  • Full JSP, ASP and HTML support: ignoring imports and comments
  • Simple GUI for browsing duplicates
  • Full Python support
  • Full VB support
  • Ada support
  • Command-line: Support wildcards in directory names
  • Strategies to optimise for speed or memory with no size limitations (over and above O/S file size limitations of course)
  • Multi-threading to take advantage of times when I/O is the bottleneck
  • Distributed processing support for large organisations that wish to perform processing across large code bases utilising multiple machines.
  • IntelliJ IDEA plugin
  • VisualStudio.NET plugin
